# Minimal VS Code Setup

A clean and minimal Visual Studio Code setup focused on:

- Minimal UI
- Smooth typography
- Better readability
- Distraction-free coding
- Kanagawa aesthetics

## Features

- Minimal interface
- Hidden scrollbars & status bar
- Compact tabs
- Thin animated cursor
- Sticky scroll improvements
- Custom italic syntax highlighting
- Optimized terminal appearance

## Theme

- Theme: August - Kanagawa
- Icons: Quill Icons

## Recommended Fonts

- JetBrains Mono
- PragmataPro Mono Liga
- MonoLisa
- Cascadia Code
- Fira Code
